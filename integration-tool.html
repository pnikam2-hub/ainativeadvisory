What happens next

  1. We review your answers and confirm scope, custom-field mappings, and sync direction.
  2. We validate API access with a test call to your tool — Phase 1: Discovery.
  3. We build and test the connector against sample data — Phase 2: Build.
  4. We deploy, run an initial full sync, and monitor for 48 hours — Phase 3: Deploy.
  5. We hand over the source code and documentation — Phase 4: Handover.

Estimated build effort: 8–12 days after this questionnaire is complete and API access is confirmed. The integration-layer code is owned by you.

What we will not do

  • Pilot before scale. Every engagement begins with a bounded pilot against one project.
  • Your schedule of record stays authoritative. Default sync is one-way into Agentic PMO; we never write back without explicit configuration.
  • No schema lock-in. The connector talks to Agentic PMO’s REST API only — no direct database writes, no backend changes.
  • Indicative, not inflated. This scopes honestly; it is not a fixed quote until API access and custom fields are confirmed.
